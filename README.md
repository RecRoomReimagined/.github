# RecRoom: Reimagined
RecRoom: Reimagined is both a revival project AND a unique spin off the classic Rec Room experience. Unlike other servers, we prioritize using older builds (2018/2019). Also unlike other server, we want to create a unique experience other revival projects don't quite create well.

# How is RecRoomReimagined unique?
We have taken an older version of Rec Room, instead of trying to revive and replicate identically the old Rec Room experience, we want to create a brand new experience and change a LOT of the way certain things work. For example, storefront items are more expensive BUT tokens and other currencies are much easier to obtain without any daily limit. Leveling and level progression has changed with a reworked system giving you xp for getting certain progression milestones (like making your first friend, creating your first room, reaching a token milestone, or a leaderboard milestone). With this, we also removed the level cap! This means that as more progression milestones and such are added, the highest possible level increases as well! There is also so much more we plan to do with this to create a more enjoyable and rewarding experience.
